# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial release of Color Sorting Game
- 7 difficulty tiers (Easy to Master)
- 45+ progressive levels with dynamic difficulty
- Difficulty selection screen at game start
- Best score tracking with localStorage persistence
- Reset best scores feature with confirmation
- Comprehensive test suite (68 tests)
- Pour animations with spring physics
- Glossy liquid visual effects with rounded segments
- Unlimited undo system
- Level restart functionality
- Return to menu option
- Move counter and best score display
- Celebration effects on level completion
- Responsive design for different screen sizes

### Technical
- React 18+ with TypeScript
- Zustand for state management
- Framer Motion for animations
- Tailwind CSS for styling
- Vitest for testing
- Dynamic tube capacity (colors + 1 formula)
- Fisher-Yates shuffle algorithm for level generation
- Guaranteed solvable level generation

## [0.1.0] - 2025-12-29

### Added
- Initial commit with full game implementation
- Production-ready codebase with CI/CD
- Comprehensive documentation
- GitHub Actions workflows
- Automated testing and linting
